### 本地预览

1：确保本地已经安装了`node.js/npm`

2：运行`npm install`，`npm run dev`然后在本地8080端口就能看到

### 编译部署

1：`npm run build`

2：将生成的dist文件夹下的文件放在服务器的tomcat中

## 项目支持
